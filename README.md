# Teste Roga Labs

### Case 1 - Get

#### Exemplo de uso
```
git clone git@github.com:thaismtss/test-rogalabs.git

cd test-rogalabs

cd get

node

const deepGet = require('./get.js')

deepGet('a', {a: 4})

```

#### Rodando os testes

Dentro de test-rogalabs/get rode:
```
npx jest --runTestsByPath get.test.js
```
